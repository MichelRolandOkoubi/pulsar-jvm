## [Unreleased]

### Added
- Initial project structure
- `pulsar-core`: ZeroAllocPool, ObjectPool, NativeBuffer, primitive collections, lock-free queues
- `pulsar-memory`: Arena/Slab/Stack allocators, off-heap map, memory tracker
- `pulsar-json`: Zero-copy JSON parser and writer
- `pulsar-http`: Zero-copy HTTP parser
- `pulsar-spring-boot-starter`: Spring Boot auto-configuration
- `pulsar-quarkus`: Quarkus extension
- `pulsar-benchmark`: JMH benchmarks
